== Description ==

A WordPress Image slider based on Nivo Slider to display image slider on post, page or theme. It's an easy, simple & responsive slider, which looks awesome in every single theme.

= Features =

* Fully Responsive 
* Unlimited Slides
* Supported in all major browsers
* Beautiful Transition Effects
* Support arrows navigation, dots arrows and thumbnails navigation
* Five premade beautiful themes

== Installation ==

Installing the plugins is just like installing other WordPress plugins. If you don't know how to install plugins, please review the option below:

* From your WordPress dashboard go to **Plugins > Add New**.
* Search for **Nivo Image Slider** in **Search Plugins** box.
* Find the WordPress Plugin named **Nivo Image Slider** by **Sayful Islam**.
* Click **Install Now** to install the **Nivo Image Slider** Plugin.
* The plugin will begin to download and install.
* Now just click **Activate** to activate the plugin.

If you still need help. visit [WordPress codex](https://codex.wordpress.org/Managing_Plugins#Installing_Plugins)
